## 🚀 Sobre o Projeto

O **Almoxcell** é um aplicativo web progressivo (PWA) desenvolvido para otimizar o gerenciamento, a busca e a retirada de peças de manutenção industrial. O sistema foi projetado sob medida para atender às demandas de organização e agilidade do setor de manutenção da **Baterias Excell**, garantindo o controle rigoroso do estoque de insumos técnicos, ferramentas e componentes utilizados no ecossistema fabril.

> 💡 **Impacto Real:** Este projeto não é apenas um exercício acadêmico. Ele foi apresentado, validado pela gerência e **atualmente encontra-se em uso ativo em produção** pela equipe de manutenção da empresa, resolvendo problemas reais de controle de inventário no dia a dia.

## ✨ Funcionalidades Principais

*   **Busca em Tempo Real (Auto-Digite):** Filtragem ultra-rápida de peças por nome ou local de armazenamento diretamente no campo de busca.
*   **Gestão de Estoque Dinâmica:** Controle visual de quantidade de peças disponíveis com destaque visual e tags de localização de fácil leitura.
*   **Histórico de Movimentações:** Registro cronológico automatizado indicando quem retirou ou alterou o status de uma peça.
*   **Design Responsivo Corporativo (Mobile-First):** Interface otimizada para smartphones (facilitando o uso direto no almoxarifado via touch screen) com adaptação para visualização em grid/colunas no Desktop.
*   **Níveis de Acesso (RBAC):** Diferenciação de permissões entre usuários comuns (operadores e mecânicos que apenas realizam retiradas) e administradores (gestores com permissões de cadastro e exclusão de itens).
*   **Identidade Visual Integrada:** Design totalmente personalizado utilizando como base a paleta de cores oficial da Baterias Excell.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando uma arquitetura moderna, leve e sem dependências pesadas, priorizando performance no carregamento:

*   **Frontend:** HTML5, CSS3 (com propriedades avançadas de CSS Grid, Flexbox e Media Queries para responsividade).
*   **Linguagem de Programação:** JavaScript (ES6+, Programação Assíncrona com Async/Await).
*   **Backend & Banco de Dados:** Firebase (Firestore Database para persistência em tempo real e Firebase Authentication para controle seguro de acesso dos funcionários).
*   **Arquitetura de Imagens:** Processamento e conversão de arquivos para strings em Base64 para otimização de armazenamento.
